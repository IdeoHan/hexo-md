## 开始

```Java
package J1;

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```
<!--more-->
### 解释

```Java
package J1; # 项目名

public class HelloWorld # 包名

public static void main(String[] args) # 主类

System.out.println("Hello World"); # 打印输出Hello Word
```
